Comment lire le code couleur

Chaque bande correspond à un chiffre ou un multiplicateur. La dernière bande (souvent or ou argent) indique la tolérance.

⬛ Noir = 0
🟫 Marron = 1
🟥 Rouge = 2
🟧 Orange = 3
🟨 Jaune = 4
🟩 Vert = 5
🟦 Bleu = 6
🟪 Violet = 7
⬜ Gris = 8
⬜ Blanc = 9
🥇 Or = ×0.1 / ±5%
🥈 Argent = ×0.01 / ±10%
