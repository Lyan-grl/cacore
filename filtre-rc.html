Calcule la fréquence de coupure d'un filtre RC passe-bas, passe-haut ou passe-bande.

Type de filtre

🔉 Passe-bas — laisse passer les basses fréquences, bloque les hautes. Utilisé pour lisser un signal ou filtrer le bruit.
🔊 Passe-haut — laisse passer les hautes fréquences, bloque les basses. Utilisé pour éliminer la composante continue (DC).

Les trois types de filtres

Passe-bas — laisse passer les fréquences sous fc. Ex : lisser un signal capteur.

Passe-haut — laisse passer les fréquences au-dessus de fc. Ex : supprimer la composante DC.

Passe-bande — laisse passer une plage autour de f0. Ex : filtre radio.
